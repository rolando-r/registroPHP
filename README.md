## Run Locally

Clone the project

```bash
  git clone https://github.com/rolando-r/registroPHP/
```

Go to the project directory

```bash
  cd registroPHP
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
